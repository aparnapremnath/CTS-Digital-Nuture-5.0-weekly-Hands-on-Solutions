# Week 01 Solutions

## 📋 Overview
This folder contains solutions for Week 01 assignments as part of the CTS Digital Nurture 5.0 program.

## 📁 Files

| File | Description |
|------|-------------|
| 1)SingletonPatternExample | Singleton Design Pattern – ensures only one instance of a class is created |
| 2)FactoryMethodPatternExample | Factory Method Design Pattern – creates objects without specifying the exact class |
| 3)EcommercePlatformSearch | Search algorithms (Linear & Binary Search) for an e-commerce product catalog |
| 4)FinancialForecasting | Recursive algorithm to calculate future financial values |
| 5)Control Structures | PL/SQL Control Structures – IF-ELSE, LOOP, WHILE, FOR loop examples |
| 6)Stored Procedures | PL/SQL Stored Procedures – creating and executing stored procedures in Oracle XE |
| 7)Setting up JUnit | JUnit 4 setup with Maven in VS Code – unit testing configuration |

## 🔍 Notes
- Built using Java (JDK 17), Maven, and VS Code
- PL/SQL exercises run on Oracle Database XE (local)
- JUnit tests configured with `maven-archetype-quickstart`
