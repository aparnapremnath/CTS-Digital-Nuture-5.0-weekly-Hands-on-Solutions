\# Exercise 1: Git Hands-On Lab



\## Overview

This exercise covers the fundamentals of Git version control — configuring 

Git on a local machine, setting a default editor, and creating a tracked 

repository with a remote on GitHub.



\## Files

| File | Description |

|------|-------------|

| welcome.txt | Sample file created and tracked as part of the Git basics walkthrough |



\## Tech Stack

\- Git 2.54.0

\- GitHub

\- Windows PowerShell



\## Notes

\- Configured global Git user name and email (`git config --global`)

\- Set Notepad as the default Git editor for commit messages

\- Initialized a local repository with `git init`

\- Created, staged, and committed `welcome.txt`

\- Connected local repo to a remote GitHub repository (`git remote add origin`)

\- Pushed local commits to GitHub with `git push -u origin master`
