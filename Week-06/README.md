Week 06 Solutions

📝 Overview

This folder contains solutions for Week 06 assignments, covering Git fundamentals — configuration, .gitignore, branching and merging, conflict resolution, and syncing with a remote repository.



📂 Files

| Folder | Description |

|--------|-------------|

| 01)GitHOL | Git configuration, default editor setup, and tracking a first file in a local/remote repo |

| 02)GitIgnore | Using .gitignore to exclude log files and folders from version control |

| 03)BranchingMerging | Creating a branch, committing changes, and merging it back into master |

| 04)ConflictResolution | Simulating and resolving a merge conflict between two branches |

| 05)CleanupPush | Verifying clean repository state and syncing final changes with the remote |



🔍 Notes

All exercises were completed using Git Bash / Windows PowerShell with GitHub as the remote repository (in place of GitLab referenced in the original lab documents). VS Code's diff viewer was used in place of P4Merge for visual comparisons.
